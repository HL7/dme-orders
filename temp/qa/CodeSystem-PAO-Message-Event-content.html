PAO Message Event Code System

This is the value set for the event element in the Message Header

This code system http://hl7.org/fhir/us/dme-orders/CodeSystem/PAO-Message-Event defines the following codes:

CodeDisplayDefinition
InitialOrder Initial OrderInitial order for Post Acute services
OrderUpdate Order UpdateUpdate to an order (may be declined)
OrderCancel Order CancelCancel an order (may be declined if done)
OrderStatus Order StatusRequest for the status of the order
